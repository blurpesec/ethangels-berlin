About

ETH Angels is part of the worldwide ETH Anti Phishing Group. We are developing a website for people to easily report crypto scams into a friendly, single point of contact.

Verified scam and phishing data will then be propogated to organisations including MetaMask, MyEtherWallet, ETH Address Lookup and EtherScam DB.

This will reduce the current need for people to report scams to multiple points of contact (or give up because it's too darn difficult) and also helps avoid duplication in the scam verification process for these organisations.

We are creating a clean, simple UI where people can…

  • Report a scam
  • View / search / filter scams

And where ETH Angels can…

  • Triage incoming scams
  • Provide data and mark a scam as verified or a false positive
  • View / search / filter scams
