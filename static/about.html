About

ETH Angels is part of the worldwide ETH Anti Phishing Group and we are developing a website for people to report crypto scams into a single point of contact.

Verified scam and phishing data will then be propogated to organisations including MetaMask, MyEtherWallet, ETH Address Lookup and EtherScam DB.

This will reduce the current need for people to report scams to multiple points of contact and also helps avoid duplication in the scam verification process for these organisations.

We want to create a clean, simple UI where people can:

  • Report a scam
  • View / search / filter scams

And where ETHangels can:

  • Triage incoming scams
  • Provide data and mark a scam as verified or a false positive
  • View / search / filter scams
